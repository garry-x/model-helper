About Benchmarks

This system tracks the following major LLM benchmarks:

  • MMLU - Massive Multitask Language Understanding (57 subjects)
  • HellaSwag - Commonsense inference for event ordering
  • ARC - AI2 Reasoning Challenge (grade-school science)
  • TruthfulQA - Truthfulness evaluation against misinformation
  • Winogrande - Coreference resolution at scale
  • GSM8K - Grade School Math problems
  • HumanEval - Code generation evaluation
  • Chatbot Arena - Human preference ranking via blind comparisons
